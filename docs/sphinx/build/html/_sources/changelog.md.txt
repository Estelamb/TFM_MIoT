# Changelog

## 0.1.0 — PoC (current)

- Initial 7-service architecture: api-gateway, device, ai, script,
  compilation, deployment, monitoring
- Hailo-8 / Hailo-8L compiler via Docker (hailo_ai_sw_suite)
- Sony IMX500 compiler via MCT + imx500-converter
- Next.js 15 frontend with Demo Mode
- Edge agent with SHA-256 verified downloads and psutil telemetry
- MQTT anonymous transport (Mosquitto)
- Prometheus metrics for device CPU/RAM (monitoring-service)
