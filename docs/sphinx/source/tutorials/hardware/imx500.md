## Introduction to the Raspberry Pi AI Camera (IMX500)

The Raspberry Pi AI Camera features the Sony IMX500 intelligent vision sensor, which processes neural networks directly on-board. This enables low-latency image processing and offloads AI workloads from the host Raspberry Pi CPU.

## Hardware Requirements

* Raspberry Pi 5.
* Raspberry Pi AI Camera (with standard RPi5/Zero CSI cable).

## Configuration and Verification

1. Connect the camera to the CSI interface of the Raspberry Pi 5.
2. Upgrade your system packages:
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```
3. Verify the camera is detected:
   ```bash
   libcamera-hello --list-cameras
   ```
   You should see a camera description matching the `imx500` sensor.
